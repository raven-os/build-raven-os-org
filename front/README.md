## Getting Started

### Prerequisites

Install **[npm](https://www.npmjs.com/get-npm)** and **[node](https://nodejs.org/en/download/package-manager/)**

### Installing

Install dependencies

```bash
$ npm install
```
### Run

#### Development

```bash
$ npm run serve
```

#### Production

Files will be located in `../static` and serve by rocket api

```bash
$ npm run build
```
